# Beta 分布

Beta 分布可以看做是一个概率的概率分布, 当

参数为 $\alpha, \beta$ 的 Beta 分布的众数是 $\frac{\alpha - 1}{\alpha + \beta - 2}$, 期望是 $\frac {\alpha} {\alpha + \beta}$, 方差是 $\frac{\alpha \beta}{(\alpha + \beta)^2 (\alpha + \beta + 1)}$.
